    Add-on Information

    Display the updated price of Bitcoin in your browser window under status bar.

    The following price sources are available:

    • MtGox
    • BitStamp
    • BTC-e
    • BitPay
    • CoinDesk
    • Coinbase
    • BTCChina
    • Mercado Bitcoin
    • BTCTurk
    • Vircurex
    • CampBX
    • BitX.co.za
    • Bitcoin Venezuela
    • Localbitcoins
    • CaVirTex
    • Bitconic.nl
    • Bitcurex

    Currencies supported:

    • US Dollar ($)
    • Euro (€)
    • Pound (£)
    • Yuan (元)
    • Yen (¥)
    • Brazilian Real (R$)
    • Turkish Lira (TRY)
    • Russian Ruble (RUR)
    • Venezuelan Bolivar (VEF)
    • Argentinian Peso (ARS)
    • Polish zloty (zŁ)
    • Canadian dollar (C$)
    • South African Rand (ZAR)
    • Chilean Peso (CLP)
    • Litecoin (Ł)
    • Worldcoin
    • Dogecoin
    • Namecoin

    Configuration options:

    • Select exchange price update interval
    • Select exchanges and currencies
    • Select ticker colors and backgrounds
    • Show short and long trend indicators
    • Customize font size and spacing
    • Select order of tickers displayed (HOW: disable all tickers and enable in order)

    Add-on Settings

    In Firefox's top menu select Addons (or Ctrl+Shift+A)->Extensions->Options
