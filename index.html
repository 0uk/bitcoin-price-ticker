    Add-on Information

    Display the updated price of Bitcoin in your browser window under status bar.

    The following price sources are available:

    • BitStamp
    • BTC-e
    • Kraken
    • BitPay
    • CoinDesk
    • Coinbase
    • BTCChina
    • Mercado Bitcoin

    Configuration options:

    • Select displaying the tickers as individual Widgets in the navigation bar (that you can drag&drop with Firefox->Open Menu->Customize) or display all tickers in a separate tool bar
    • Select exchange price update interval
    • Select exchanges and currencies
    • Select ticker colors and backgrounds
    • Show currency name (BTC, ETH, LTC, etc.) and disable rounding of decimals
    • Customize font size
    • Select order of tickers displayed (in navigation bar use Firefox->Open Menu->Customize and drag&drop. In separate toolbar disable all tickers and enable in order from Addon options)

    Add-on Settings

    In Firefox's top menu select Addons (or Ctrl+Shift+A)->Extensions->Options
